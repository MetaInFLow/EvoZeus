- Status: active
- Last updated: 2026-06-15

## 这里放什么

`start/` 放第一次使用、最短链路和可跟着走的教程。读者应该能完成一个动作，并在结尾看到明确结果。

报告类型放在 [../reports/README.zh-CN.md](../reports/README.zh-CN.md)。概念解释放在 [../concepts/](../concepts/)。短术语定义放在 [../glossary/README.zh-CN.md](../glossary/README.zh-CN.md)。协议、schema、模板放在 [../reference/](../reference/)。

## Fastest Path

| Step | Do | Result |
| --- | --- | --- |
| 1 | 把启动语发给 Agent | Agent 读取 `SKILL.md` |
| 2 | 等 Agent 输出 Verdict Card | 看到 evidence、signals、verdict、privacy note |
| 3 | 按报告选择下一步 | 保存、修正、贡献、忽略或继续观察 |
