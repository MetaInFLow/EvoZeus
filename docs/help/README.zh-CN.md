## 常见判断

- evidence 不足：标记为 `Open Case`，继续观察。
- verdict 太跳：要求 Agent 补 evidence。
- 报告包含敏感信息：先做 redaction review。
- 因子依赖不清楚：先 inspect manifest。
- 社区贡献不确定：保存本地草稿。
