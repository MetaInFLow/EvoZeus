# EvoZeus Redaction

Use this skill before anything becomes public.

## When to Use

- Preparing public issue or PR content.
- Sharing report excerpts.
- Creating examples from real sessions.
- Contributing Cases, Rules, Factors, or Golden Cases.
- Checking whether a report can leave local context.

## Remove or Generalize

- Names, emails, phone numbers, account IDs.
- Tokens, secrets, API keys, credentials.
- Customer names, internal project names, private paths.
- Unreleased code details.
- Business-sensitive context.

## Keep

- Redacted evidence summary.
- Scenario.
- Proposed verdict.
- Reusable rule or factor signal.
- Privacy note.

## Rule

Raw private sessions do not leave local context by default.
