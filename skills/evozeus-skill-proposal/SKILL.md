# EvoZeus Skill Proposal

Use this skill when evidence suggests a new scenario skill may be useful.

## When to Use

- The user repeatedly corrects the Agent in the same scenario.
- A failure repeats across sessions.
- An adhoc approach creates unusually high-quality output.
- A local rule starts becoming reusable.

## Proposal Shape

```text
# Skill Proposal: <scenario>

## Scenario
## Evidence
## Repeated Correction or Success Pattern
## Proposed Skill Behavior
## Inputs
## Outputs
## Permissions
## Privacy Notes
## Validation Scenario
```

## Rule

Do not recommend generic skills. Propose a scenario skill only when evidence shows a repeated or high-value pattern.
