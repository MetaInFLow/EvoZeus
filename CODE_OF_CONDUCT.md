# Code of Conduct

EvoZeus 讨论必须围绕证据、可复现行为和可维护规则展开。

## Expected Behavior

- 用证据说明问题，不攻击个人。
- 尊重隐私边界，不公开 raw session、客户数据、密钥、私有路径或未发布业务上下文。
- 对 AI-assisted work 保持透明，说明 agent 做了什么、哪些检查由本地环境或维护者验证。
- 接受 maintainer 对 PR 拆分、补 evidence、补 redaction、转 RFC 或关闭低信号贡献的决定。

## Unacceptable Behavior

- 上传敏感数据或要求他人公开敏感数据。
- 用大面积混改绕过 review。
- 在没有 evidence 的情况下反复提交 opinion-only Candidate。
- 人身攻击、骚扰、歧视或威胁。

## Enforcement

维护者可以要求修改、隐藏内容、关闭 issue/PR、限制参与，或将安全问题转入私下处理。
